Калькулятор может производить операции как с рациональными, так и с комплексными числами.
Комплексное число нужно записывать в формате n+ki (например 6-2i первое и второе число).
Рациональные числа для универсальности нужно записывать в формате: 
- первыое число х/1 (например 10/1 или -10/2) ;
- второе число х/1 (например 10/1 или -10/2); 
Операциями являются:
+ -> сложить
- -> отнять
* -> умножить
/ -> разделить
Список операций выводится одним из 3-ёх дейстий решения, вводить его в поле к первому или второму числу не требуется. 
Для запуска используйте файл z_2.py
Далее для старта программы введите в боте /start
Для запроса файла с логами введите в боте /start и далее команду /log
После запроса лога данных необходимо перезапустить программу через терминал.
